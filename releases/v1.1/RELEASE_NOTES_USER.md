# v1.1 升级公告（用户版）

## 版本定位
v1.1 在 v1.0 基础上，重点提升了「账号能力」「管理效率」「移动端体验」和「分享填写流程」。

## 主要升级点

1. 账号与权限升级
- 新增注册、登录、退出、修改密码。
- 区分管理员与普通用户角色，页面与功能按权限展示。

2. 后台管理更高效
- 新增按状态、分类、关键词筛选。
- 新增统计看板，快速查看发布量、可见量、置顶和过期情况。
- 可直接查看每个表单的数据并导出。

3. 专用填写链接
- 管理员可为单个表单生成专用链接。
- 用户可通过链接直达填写页面，无需先进入表单列表。

4. 表单展示与排序优化
- 首页支持按业务分类展示。
- 支持置顶、发布状态、优先级、发布时间、过期时间等排序规则。

5. 移动端体验增强
- 首页与管理页针对手机竖屏做了专门适配。
- 列表、筛选、操作按钮在小屏下可用性明显提升。

6. 稳定性与维护性提升
- 配置支持拆分为多个 YAML 文件统一加载。
- 数据统一入库，减少历史双写带来的维护复杂度。

## 兼容性与注意事项
- v1.1 保持现有业务数据可继续使用。
- 建议升级后检查一次表单状态与过期时间配置。
- 生产环境建议修改默认管理员密码，并配合 HTTPS 使用。

## 一句话总结
v1.1 从“可用”迈向“可运营”，既增强了后台管理能力，也显著优化了填写端体验。
